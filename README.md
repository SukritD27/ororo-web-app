# Build/Run Instructions

## Build Instructions
1. npm i
2. Create a .env file in the application folder and have the following variable names:
    DB_HOST, 
    DB_USER, 
    DB_PASSWORD,  
    DB_NAME

3. npm run builddb

## Run Instructions
1. cd into the applications folder
2. npm start
